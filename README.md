# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn data to identify the key factors influencing customer attrition and provides actionable business recommendations. The project demonstrates an end-to-end data analytics workflow using Python, SQL Server, and Power BI.

---

## 🎯 Business Problem
Customer churn directly impacts business revenue. The objective of this project is to identify:
- Which customers are most likely to churn.
- Key reasons behind customer churn.
- Customer segments with the highest churn rates.
- Business strategies to improve customer retention.

## 📈 Key Insights

- Overall churn rate: **26.54%**
- Month-to-month customers have the highest churn.
- Fiber optic customers show the highest churn among internet service types.
- Competitor offers are the leading reason customers leave.
- Long-term contracts significantly improve customer retention.

---

## 💼 Business Recommendations

- Introduce loyalty offers for month-to-month customers.
- Improve customer support for Fiber optic users.
- Launch competitive pricing campaigns.
- Encourage customers to switch to annual contracts.
